### Setup and Configuration

1. **React and Other Version Dependencies**: View the detailed list of dependencies [here](https://github.com/okto-hq/okto-sdk-react-template-app/blob/main/package.json).

2. **Getting the API Key**: Go to the [dashboard](https://dashboard.okto.tech/) to get the API Key for the Okto SDK.

3. **Getting the Google Client ID**: Follow steps [here](/docs/react-sdk/advanced-sdk-config/authenticate-users/google-oauth/learn) to setup a new Google Console Project and get your Google Client ID.

<Callout title="Default Environment">
    The default environment configuration is **Sandbox**. If you want to upgrade to **Production**, you can check out how to upgrade in [Admin Panel](/docs/react-sdk/developer-admin-dashboard/learn).
</Callout>
