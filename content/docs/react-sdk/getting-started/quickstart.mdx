### Installation

Install the latest version of the Okto React SDK using your package manager of choice:

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab value="npm">
  ```bash
    npm i okto-sdk-react
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm add okto-sdk-react
  ```
  </Tab>
  <Tab value="yarn">
   ```bash
  yarn add okto-sdk-react
  ```
  </Tab>    
</Tabs>

### Initializing the SDK

You need to first intialize the SDK with the API key to get started. You can do this in the root file of your app, for example `App.tsx` or `index.tsx`.

```typescript title="App.tsx"
import { OktoProvider, BuildType } from 'okto-sdk-react-native';
 
<OktoProvider apiKey={OKTO_CLIENT_API} buildType={BuildType.SANDBOX}>
    <App />
</OktoProvider>
```

### Existing Project setup

1. Ensure your React version is compatible. (Refer [here](link-to-package-json) for list of all dependencies).

2. **Web3 Projects** (Migration coming soon). Visit the [Migration Section](link) here.

3. **Web2 projects**
    - Wrap your project in the `Okto Provider` along with your API key.
    ```typescript title="App.tsx"
        import { OktoProvider, BuildType } from 'okto-sdk-react-native';
 
        <OktoProvider apiKey={OKTO_CLIENT_API} buildType={BuildType.SANDBOX}>
            <App />
        </OktoProvider>
    ```
