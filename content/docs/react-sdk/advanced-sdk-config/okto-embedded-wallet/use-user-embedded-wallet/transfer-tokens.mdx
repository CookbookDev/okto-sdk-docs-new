## Transfer tokens

<sub><i>async</i></sub> `transferTokens(data)`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L344" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> transfers tokens based on the provided data.

### Parameters

| Parameter   | Type            | Description                                     | Possible Values |
|-------------|-----------------|-------------------------------------------------|-----------------|
| `data`     | `TransferTokens` | Data being provided for transferring the tokens | -               |

```typescript
interface TransferTokens {
  network_name: string;
  token_address: string;
  quantity: string;
  recipient_address: string;
}
```

### Response

<Callout title="Success Response">

| Field Name | Type                          | Description                                      | Possible Values |
|------------|-------------------------------|--------------------------------------------------|-----------------|
| `result`   | `Promise<TransferTokensData>` | Returns information about the tokens transferred | -               |

<Accordions>
<Accordion title="Response Type">
```typescript
interface TransferTokensData {
  orderId: string;
}
```
</Accordion>
</Accordions>

</Callout>



<Callout type="error" title="Error Response">

| Field Name | Type     | Description                                     | Possible Value|
|------------|---------------|--------------------------------------------|---------------|
| `error`    | `error`       |  Error on unsuccessful execution           | Error details |

</Callout>


<Accordions>
<Accordion title="Example">
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
        const { transferTokens } = useOkto() as OktoContextType;
 
        const networkName = "POLYGON_TESTNET"
        const tokenAddress = "x2f7b97837f2d14ba2ed3a4b2282e259126a9b848"
        const quantity = "1"
        const recipientAddress = "0x0FC096A53343C68c1086e626661A4bb31486A9e2"
 
        transferTokens({
            network_name: networkName,
            token_address: tokenAddress,
            recipient_address: recipientAddress,
            quantity: quantity
        }).then((result) => {
                console.log('Transfer success', result);
        })
        .catch((error) => {
            console.log('Transfer error', error);
        });
    ```
  </Tab>
</Tabs>
</Accordion>
</Accordions>

## Fetch order history

<sub><i>async</i></sub> `orderHistory(query)`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L315" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> fetches the order history based on the provided query.

### Parameters

| Parameter   | Type                  | Description                      | Possible Values |
|-------------|-----------------------|----------------------------------|-----------------|
| `query`     | `Partial<OrderQuery>` | Query for fetching order history | Auth idToken    |

```typescript
interface OrderQuery {
  offset: number;
  limit: number;
  order_id: string;
  order_state: string;
}
```

</Accordion>
</Accordions>

</Callout>

<Callout type="error" title="Error Response">

| Field Name | Type     | Description                                     | Possible Value|
|------------|---------------|--------------------------------------------|---------------|

</Callout>


<Accordions>
<Accordion title="Example">
<Tabs items={['Typescript']}>

    ```typescript

        const { orderHistory } = useOkto() as OktoContextType;
 
        orderHistory({order_id: orderId})
        .then((orders) => {
            const order = orders.find((order: any) => order.order_id === orderId);
            if(order) console.log(order)

        .catch((error) => {
            console.error(`error:`, error);
    ```
