## Get all supported networks

`getSupportedNetworks()` retrieves all the whitelisted networks on your API Key.

### Parameters

There are no parameters for this function.

### Response

<Callout title="Success Response">

| Field Name | Type                   | Description                              | Possible Values |
|------------|------------------------|------------------------------------------|-----------------|
| `result`   | `Promise<NetworkData>` | Returns list of whitelisted networks     | -               |

</Callout>



<Callout type="error" title="Error Response">

| Field Name | Type                  | Description                                | Possible Value|
|------------|-----------------------|--------------------------------------------|---------------|
| `error`    | `error`               |  Error on unsuccessful execution           | Error details |

</Callout>


### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType, type User} from 'okto-sdk-react';
 
        const { getSupportedNetworks } = useOkto() as OktoContextType;
 
        getSupportedNetworks()
            .then((result) => {
                console.log(result)
            })
            .catch((error) => {
                console.error(`error:`, error);
            });
    ```
  </Tab>
</Tabs>
