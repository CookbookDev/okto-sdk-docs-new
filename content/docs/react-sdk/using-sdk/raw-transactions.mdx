## Get raw transactions

`getRawTransactionStatus(query: RawTransactionStatusQuery)` fetches the status of a raw transaction based on the provided query.

### Parameters

| Parameter   | Type                        | Description                            | Possible Values |
|-------------|-----------------------------|----------------------------------------|-----------------|
| `query`     | `RawTransactionStatusQuery` | Query being provided fetching the data | -               |


```typescript
getRawTransactionStatus: ({order_id: string}) => Promise<RawTransactionStatusData>;
```

### Response

<Callout title="Success Response">

| Field Name | Type                                | Description                                             | Possible Values |
|------------|-------------------------------------|---------------------------------------------------------|-----------------|
| `result`   | `Promise<RawTransactionStatusData>` | Returns information about the status of raw transaction | -               |

</Callout>

<Callout type="error" title="Error Response">

| Field Name | Type     | Description                                     | Possible Value|
|------------|---------------|--------------------------------------------|---------------|
| `error`    | `error`       |  Error on unsuccessful execution           | Error details |

</Callout>


### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
        const { getRawTransactionStatus } = useOkto() as OktoContextType;
 
        const networkName = "POLYGON_TESTNET"
        const from = "0x0FC096A53343C68c1086e626661A4bb31486A9e2"
        const to = "0x0FC04889A533C68c1086e626661A4bb31486A9e2"
        const data = "transaction";
        const value = "1";
 
        getRawTransactionStatus({
            network_name: networkName,
            transaction: {
                from,
                to,
                data,
                value,
            }
        }).then((result) => {
                console.log('Transfer success', result);
        })
        .catch((error) => {
            console.log('Transfer error', error);
        });
    ```
  </Tab>
</Tabs>
