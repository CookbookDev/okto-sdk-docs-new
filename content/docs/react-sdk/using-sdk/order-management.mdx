## Fetch order history

`orderHistory(query: Partial<OrderQuery>)` fetches the order history based on the provided query.

### Parameters

| Parameter   | Type         | Description                      | Possible Values |
|-------------|--------------|----------------------------------|-----------------|
| `query`     | `OrderQuery` | Query for fetching order history | Auth idToken    |

```typescript
interface OrderQuery {
  offset: number;
  limit: number;
  order_id: string;
  order_state: string;
}
```

### Response

<Callout title="Success Response">

| Field Name | Type                 | Description                                 | Possible Values |
|------------|----------------------|---------------------------------------------|-----------------|
| `result`   | `Promise<OrderData>` | Returns information about the order history | -               |

</Callout>



<Callout type="error" title="Error Response">

| Field Name | Type     | Description                                     | Possible Value|
|------------|---------------|--------------------------------------------|---------------|
| `error`    | `error`       |  Error on unsuccessful execution           | Error details |

</Callout>


### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType, type Order} from 'okto-sdk-react';
 
        const { orderHistory } = useOkto() as OktoContextType;
 
        orderHistory({order_id: orderId})
        .then((orders) => {
            const order = orders.find((order: any) => order.order_id === orderId);
            if(order) console.log(order)
        })
        .catch((error) => {
            console.error(`error:`, error);
        });
    ```
