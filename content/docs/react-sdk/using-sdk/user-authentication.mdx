## Check if the User is Logged In

`isLoggedIn` checks if the user is currently logged in.

### Parameters

There are no parameters for this function.

### Response

<Callout title="Success Response">

| Field Name | Type     | Description                            | Possible Values |
|------------|---------------|----------------------------------------|-----------------|
| `result`   | `boolean`     | Results true if the user is logged in  | `true`, `false` |

</Callout>



<Callout type="error" title="Error Response">

| Field Name | Type     | Description                                     | Possible Value|
|------------|---------------|--------------------------------------------|---------------|
| `error`    | `error`       |  Error on unsuccessful execution           | Error details |

</Callout>


### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
    import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
    const { isLoggedIn } = useOkto() as OktoContextType;
    isLoggedIn().then(result => console.log(isLoggedIn ? "User is logged in" : "User is logged out"));
```
  </Tab>
</Tabs>

## Authenticate a User

`authenticate` authenticates a user using an ID token.

### Parameters
| Parameter   | Type | Description                                               | Possible Values |
|-------------|-----------|------------------------------------------------------|-----------------|
| `idToken`   | `string`  | Token received from Google OAuth2                    | Auth idToken    |
| `callback`  | `function`| Callback function to be called after authentication. | -               |

### Response

<Callout title="Success Response">

| Field Name | Type | Description                               | Possible Values |
|------------|-----------|--------------------------------------|-----------------|
| `result`   | `any`     | Result object upon successful auth.  | Auth details    |

</Callout>

<Callout type="error" title="Error Response">

| Field Name | Type | Description                               | Possible Values |
|------------|-----------|--------------------------------------|-----------------|
| `error`    | `any`     | Error object on authentication fail. | Error details   |

</Callout>

### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
    import {
      useOkto,
      type OktoContextType,
    } from 'okto-sdk-react';
 
    const { authenticate } = useOkto() as OktoContextType;
    const idToken = 'USER_ID_TOKEN';
    authenticate(idToken, (result, error) => {
      if (result) {
          console.log('authentication successful');
      }
      if (error) {
          console.error('authentication error:', error);
      }
  });
```
  </Tab>
</Tabs>

## Log Out

`logOut` logs out the current user from the session.

### Parameters

There are no parameters for this function.

### Response 

<Callout title="Success Response">
| Field Name | Type                   | Description                               | Possible Values |
|------------|------------------------|-------------------------------------------|-----------------|
| `result`   | `Promise<Boolean>`     | Returns `true` if user is logged out      | `true`          |
| `error`    | `Promise<Boolean>`     | Returns `false` if user is not logged out | `false`         |

</Callout>


<Callout title="Error Response" type="error">
| Field Name | Type                   | Description                               | Possible Values |
|------------|------------------------|-------------------------------------------|-----------------|
| `error`    | `Promise<Error>`       | Returns 'error' on unsuccessful logout    | Error details   |
</Callout>


### Example
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
   ```typescript
  import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
  const { logout } = useOkto() as OktoContextType;
  logout().then(result => console.log("logged out"));
  ```
  </Tab>
</Tabs>
