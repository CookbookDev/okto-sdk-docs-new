### Methods Overview

- [`isLoggedIn`](#check-if-the-user-is-logged-in): Check if the user is currently logged in.
- [`authenticate`](#authenticate-a-user): Authenticate a user using an ID token.
- [`logOut`](#log-out): Log out the current user

## Check if the User is Logged In

`isLoggedIn` checks if the user is currently logged in.

### Parameters

There are no parameters for this function.

#### Success Response

| Field Name | Data Type     | Description                            | Possible Value |
|------------|---------------|----------------------------------------|----------------|
| `result`   | `boolean`     | Results true if the user is logged in  | `true`         |

#### Error Response

| Field Name | Data Type     | Description                                | Possible Value|
|------------|---------------|--------------------------------------------|---------------|
| `error`    | `boolean`     | Results false if the user is not logged in | `false`       |

#### Example
```typescript
import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
const { isLoggedIn } = useOkto() as OktoContextType;
isLoggedIn().then(result => console.log(isLoggedIn ? "User is logged in" : "User is logged out"));
```

## Authenticate a User

`authenticate` authenticates a user using an ID token.

#### Parameters
<TypeTable
  type={{
    idToken: {
      description:
        'The ID token for the user',
      type: 'string',
      default: null,
    },
   callback: {
      description:
        'Callback function for the result',
      type: 'function',
      default: null,
    },
  }}
/>


#### Success Response

| Field Name | Data Type | Description                          | Optional? | Default Value | Possible Values |
|------------|-----------|--------------------------------------|-----------|---------------|-----------------|
| `result`   | `any`     | Result object upon successful auth.  | No        | N/A           | Auth details    |

#### Error Response

| Field Name | Data Type | Description                          | Optional? | Default Value | Possible Values |
|------------|-----------|--------------------------------------|-----------|---------------|-----------------|
| `error`    | `any`     | Error object on authentication fail. | No        | N/A           | Error details   |

#### Example
```typescript
import {
  useOkto,
  type OktoContextType,
} from 'okto-sdk-react';
 
const { authenticate } = useOkto() as OktoContextType;
const idToken = 'USER_ID_TOKEN';
authenticate(idToken, (result, error) => {
    if (result) {
        console.log('authentication successful');
    }
    if (error) {
        console.error('authentication error:', error);
    }
});
```

## Log Out

`logOut` logs out the current user from the session.

### Parameters

There are no parameters for this function.

#### Success Response

| Field Name | Data Type              | Description                          | Possible Values |
|------------|------------------------|--------------------------------------|-----------------|
| `result`   | `Promise<Boolean>`     | Returns `true` if user is logged out | `true`          |

#### Error Response

| Field Name | Data Type              | Description                               | Possible Values |
|------------|------------------------|-------------------------------------------|-----------------|
| `error`    | `Promise<Boolean>`     | Returns `false` if user is not logged out | `false`         |

#### Example
```typescript
import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
const { logout } = useOkto() as OktoContextType;
logout().then(result => console.log("logged out"));
```

</div>
