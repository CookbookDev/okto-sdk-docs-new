## General Errors

| **Code**                 | **Message**                              | **Description**                                                                                                                                       |
|--------------------------|------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| `BAD_REQUEST_ERROR`          | Invalid request.                            | The server could not understand the request due to malformed syntax. Please check the request and try again.                                          |
| `DATA_NOT_FOUND_ERROR`       | Data not found.                             | The requested data could not be found. Ensure that the correct parameters are provided.                                                              |
| `DB_ERROR`                   | Database error.                             | An unexpected error occurred with the database. Please try again later. If the issue persists, contact support.                                       |
| `INVALID_DATA_ERROR`         | Invalid data provided.                      | The data provided in the request is not valid. Ensure that all required fields are correctly filled out and try again.                                |
